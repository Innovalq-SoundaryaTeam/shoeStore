# SoleHaus — Shoe Store Static Website

A complete 11-page static website for a shoe and footwear store, built with HTML5, CSS3, Bootstrap 5, and vanilla ES6+ JavaScript.

## Quick Start
Open `pages/index.html` in any modern browser, or serve the folder:

```bash
cd shoe-store
python3 -m http.server 8000
# Visit http://localhost:8000/pages/index.html
```

## Features
- 11 fully-built pages (Home, About, Products, Product Details, Brands, Size Guide, Blog, Blog Details, Contact, 404, Coming Soon)
- Working product search (navbar) with filtering on the products page
- Working cart with localStorage + badge counter
- Dark/Light mode toggle (persisted)
- RTL support (set `<html dir="rtl">`)
- Form validation (contact + newsletter)
- Countdown timer (coming-soon page)
- Responsive (mobile / tablet / desktop)
- Real shoe imagery from Unsplash CDN
- SEO: unique titles/descriptions, JSON-LD, sitemap.xml, robots.txt
